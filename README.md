Project Title: HR Analytics Dashboard

Project Description: Developed an interactive HR Analytics Dashboard in Power BI to analyze workforce trends and attrition patterns across departments. The dashboard visualizes key HR metrics such as attrition rate, employee demographics, average age, salary distribution, and job satisfaction. It provides clear insights into factors affecting employee turnover, helping HR teams make data-driven decisions on retention strategies and workforce planning.

Purpose of the Project: The goal of this project was to identify the key reasons behind employee attrition and analyze workforce composition across different categories such as gender, age, education, and experience. By visualizing these patterns, the dashboard aims to help HR departments:

Monitor attrition trends across roles and departments.

Evaluate the impact of salary, age, and tenure on attrition.

Improve employee retention strategies through data insights.

Project Learnings: Gained hands-on experience in Power BI dashboard development, including data cleaning, transformation, and DAX calculations. Learned how to design data models and create interactive visualizations that highlight key KPIs.Improved understanding of HR metrics and how analytical insights can support organizational decision-making.Enhanced storytelling skills by translating raw HR data into meaningful business insights.
